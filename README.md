## Profile for AMD reviewers

I build practical systems that help teams turn complex workflows into maintainable software. My recent work focuses on TypeScript/Node.js full-stack development, data-driven product design, CI/CD automation, and using AI-assisted development practices to improve delivery quality.

I am especially interested in roles that combine software engineering with cross-functional collaboration: understanding domain experts' pain points, translating them into clear system requirements, and building reliable tools that make engineering workflows easier to operate, measure, and improve.

## Engineering values

- Clarify the real workflow problem before implementing.
- Keep system behavior observable through tests, documentation, and automation.
- Prefer simple, maintainable designs that non-specialists can understand.
- Use AI tools as engineering accelerators, not replacements for review, tests, and accountability.
